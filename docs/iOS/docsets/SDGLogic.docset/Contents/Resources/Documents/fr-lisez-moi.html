SDGLogic

SDGLogic („SDGLogique“) étend le support pour la logique de Boole.

Ὁ δέ, οὐ μαίνομαι, φησί, κράτιστε Φῆστε, ἀλλὰ ἀληθείας καὶ σωφροσύνης ῥήματα ἀποφθέγγομαι.
Je ne suis pas fou, très excellent Festus. Je prononce des paroles de vérité et de bon sens.
                                                                                                    ―‎שאול/Shaʼul

Fonctionnalités

(L’IPA française n’a pas encore été corrigée par un locuteur natif et peut contenir des fautes d’orthographe. Des suggestions d’améloiration serait bienvenues.)

  • Opérateurs: ≠, ¬, ∧, ∨, etc.

Importation

SDGLogic est prévu pour utilisation avec le Swift Package Manager.

Ajoutez SDGLogic simplement dans la liste des dépendances dans Package.swift :

let package = Package(
    ...
    dependencies: [
        ...
        .Package(url: "https://github.com/SDGGiesbrecht/SDGLogic", versions: "1.1.0" ..< "2.0.0"),
        ...
    ]
)

Puis SDGLogic peut être importé dans des fichiers sources :

import SDGLogic

Example d’utilisation

import SDGLogic

func essayerOpérateurs() {
    si(¬((vrai ∧ faux) ∨ vrai) ≠ faux) {
        print("Je suis embrouillé...")
    }
}
