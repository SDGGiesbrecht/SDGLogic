SDGLogic

SDGLogic extends support for Boolean logic.

SDGLogic provides:

  • Operators: ≠, ¬, ∧, ∨, etc.

(For a list of other modules in the SDG family, see here.)

Importing

SDGLogic is intended for use with the Swift Package Manager.

Simply add SDGLogic as a dependency in Package.swift:

let package = Package(
    ...
    dependencies: [
        ...
        .Package(url: "https://github.com/SDGGiesbrecht/SDGLogic", versions: "1.1.0" ..< "2.0.0")
    ]
)

SDGLogic can then be imported in source files:

import SDGLogic

Usage Example

import SDGLogic

func tryOperators() {
    if ¬((true ∧ false) ∨ true) ≠ true {
        print("I’m confused...")
    }
}
